# 歡迎來到我的部落格！

這是我使用 Astro 5 建立的全新個人網站的第一篇文章。

## 技術棧

這個網站使用了以下技術：

- **Astro 5** - 現代靜態網站生成器
- **TypeScript** - 型別安全的 JavaScript
- **Tailwind CSS** - 實用優先的 CSS 框架
- **MDX** - Markdown + React 元件

## 特色功能

- 🌙 深色模式支援
- 📱 完全響應式設計
- ⚡ 極快的載入速度
- 🎨 簡約優雅的設計
- 📝 Content Collections 內容管理
